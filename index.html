Declaring Variables

You can declare a variale in hree ways:

With the keywords var. For example,

Var x = 42.

This syntaxe can be used to declare both local and global variales.

By simply assigning it a value.For example:

x = 42.

This always declares a global variable.It generates a shrict Javascript warning.

With the keyword let. For example:

let y = 13.

This syntax can be used to declare a block scope local variable. See variable scope bellow.
